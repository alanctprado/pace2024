MLinExpr.getValue()

getValue ( )

Compute the value of a linear matrix expression using the current solution.

Return value:

Value of expression as an ndarray.

Example usage:

  expr = A @ x + b
  model.addConstr(expr == 0)
  model.optimize()
  val = expr.getValue()
