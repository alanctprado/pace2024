MLinExpr.__le__()

__le__ ( )

Overloads the <= operator, creating a TempConstr object that captures an array of inequality constraints. The result is typically immediately passed to Model.addConstr.

Return value:

A TempConstr object.

Example usage:

  m.addConstr(A @ x <= 1)
