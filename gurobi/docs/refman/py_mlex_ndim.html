MLinExpr.ndim

ndim

The number of dimensions in this expression.

Return value:

An int

Example usage:

  expr1 = 2 * model.addMVar((3,)) + 1
  print(expr1.ndim)  #  "1"
  expr2 =  2 * model.addMVar((1, 3)) + 1
  print(expr2.ndim)  #  "2"
