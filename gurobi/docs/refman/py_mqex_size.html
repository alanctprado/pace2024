MQuadExpr.size

size

The total number of elements in this expression.

Return value:

An int

Example usage:

  expr1 = 2 * model.addMVar((3,))**2 + 1
  print(expr1.size)  #  "3"
  expr2 = 2 * model.addMVar((2, 3))**2 + 1
  print(expr2.size)  #  "6"
