MQuadExpr.zeros()

zeros ( shape )

Construct an all-zero MQuadExpr object of given shape.

Arguments:

shape: An int, or tuple of int. The requested shape.

Return value:

An MQuadExpr, initialized to zero.

Example usage:

  mle = gp.MQuadExpr.zeros(3)
  x = model.addMVar(3)
  mle += 2 * x**2
