MVar.T

T

Synonymous property for transpose.

Example usage:

  x = model.addMVar((4, 1))  # Resembles a "column vector"
  x_t = x.T  # Same variables, but as a "row vector"
  y = model.addMVar((3, 2))
  y_t = x.T  # Has shape (2, 3)
