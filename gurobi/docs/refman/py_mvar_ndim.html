MVar.ndim

ndim

The number of dimensions in this matrix variable.

Return value:

An int

Example usage:

  x1 = model.addMVar((3,))
  print(x1.ndim)  #  "1"
  x2 = model.addMVar((1, 3))
  print(x2.ndim)  #  "2"
