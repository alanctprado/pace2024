MVar.fromvar()

fromvar ( var )

Convert a Var object into a 0-dimensional MVar object.

Arguments:

var: The variable object to populate the returned MVar.

Return value:

MVar object corresponding to the input variable.

Example usage:

  x = model.addVar()
  x_as_mvar = MVar.fromvar(x)
