MLinExpr.shape

shape

The shape of this expression.

Return value:

A tuple of int

Example usage:

  expr1 = 2 * model.addMVar((3,)) + 1
  print(expr1.shape)  #  "(3,)"
  expr2 = 2 * model.addMVar((1, 3)) + 1
  print(expr2.shape)  #  "(1, 3)"
