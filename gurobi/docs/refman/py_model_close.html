Model.close()

close ( )

Free all resources associated with this Model object. This method is a synonym for dispose.

After this method is called, this Model object must no longer be used.

Example usage:

  env = Env()
  model = read("misc07.mps", env)
  model.optimize()
  model.close()
  env.close()
