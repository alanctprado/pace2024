MVar.size

size

The total number of elements in this matrix variable.

Return value:

An int

Example usage:

  x1 = model.addMVar((3,))
  print(x1.size)  #  "3"
  x2 = model.addMVar((2, 3))
  print(x2.size)  #  "6"
